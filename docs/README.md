# GitHub Pages — Key Numbers (static, no backend)

This folder is the **static frontend** for the Cash TAM key numbers.

- **index.html** — Single page that loads `dashboard_data.json` and displays Organic + TAM metrics. No server, no API: all data is in the JSON file.
- **dashboard_data.json** — Generated by `export_dashboard_data.py` (run from project root). Re-run the export whenever you want to refresh the numbers, then commit and push.

## Enable the live site

1. Push the repo to GitHub (from your machine):  
   `cd /Users/priyasingh/Desktop/Interactive && bash push_to_github.sh`
2. On GitHub: **Settings → Pages → Build and deployment**
   - **Source:** Deploy from a branch
   - **Branch:** `main` → **Folder:** `/docs`
   - Save. The site will be at:  
     **https://priyasingh1995.github.io/cash_tam/**

## Updating the numbers

From project root:

```bash
python export_dashboard_data.py   # or: .venv/bin/python export_dashboard_data.py
git add docs/dashboard_data.json
git commit -m "Update dashboard snapshot"
git push
```

GitHub Pages will serve the updated JSON and the page will show new numbers on refresh.
